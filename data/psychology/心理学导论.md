# 心理学导论

## 1. 什么是心理学
心理学是对**心理与行为**进行科学研究的学科。入门学习时，可以把它理解为：研究人和动物如何感知、学习、记忆、思考、情绪体验并表现出行为的科学。  
从现代学科结构看，心理学既关注个体内部的认知与情绪过程，也关注这些过程与社会环境、发展阶段和生物基础之间的关系。

## 2. 心理学研究什么
心理学常见研究对象包括：

- **感觉与知觉**：人如何接收并组织外界信息
- **学习与记忆**：经验怎样改变行为，信息怎样被保持和提取
- **思维与语言**：概念形成、推理、判断、问题解决
- **情绪与动机**：人为什么会产生情绪、需求和目标追求
- **人格与个体差异**：不同人为什么会表现出较稳定的行为模式
- **发展与社会行为**：个体如何在成长和社会互动中变化

## 3. 心理学的主要研究方法
### 3.1 观察法
在较自然的情境下记录行为表现，适合描述现象，但较难精确控制变量。

### 3.2 实验法
通过操纵自变量并控制其他条件，观察因变量变化，是检验因果关系的重要方法。

### 3.3 调查法
通过问卷、访谈等方式收集大量样本信息，适合了解态度、意见和群体特征。

### 3.4 测验法
借助标准化工具测量智力、人格、兴趣或心理状态，强调信度与效度。

## 4. 心理学的几个核心分支
- **普通心理学**：研究心理过程的一般规律
- **发展心理学**：研究个体从婴儿到老年的心理变化
- **社会心理学**：研究个体在社会情境中的行为与认知
- **教育心理学**：研究学习、教学和学校情境中的心理问题
- **临床与咨询心理学**：关注心理评估、心理问题识别与干预
- **生物心理学**：关注脑、神经系统和行为之间的关系

## 5. 学心理学时要注意什么
1. **区分科学心理学与日常经验判断**  
   日常“我觉得”“他应该是”并不能代替系统证据。

2. **区分描述、解释和干预**  
   描述现象并不等于已经找到原因；找到原因也不等于立刻能提出有效干预。

3. **注意伦理边界**  
   心理学材料常涉及隐私、心理健康和社会评价，使用时应避免把教学分类直接当作临床诊断。

## 6. 入门知识框架建议

→ 学习

→ 思维与语言  
→ 情绪与动机  
→ 人格  
→ 发展心理学  
→ 社会心理学
