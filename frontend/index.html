标准题面

Title

牛顿第二定律应用

Description

给定一个物体的质量和它所受的力，根据牛顿第二定律 F = ma 计算该物体的加速度。

Input

两行。第一行输入质量 m，第二行输入作用力 F。

Output

一行输出加速度 a，结果保留两位小数。

Sample Input

3 6

Sample Output

2.00

参考答案与验证

reference_solution.py python 
m = float(input().strip())
F = float(input().strip())
a = F / m
print(f"{a:.2f}")
